# Hydropathy-of-amino-acids-determining-transmembrane-helises-of-a-protein
This code is designed to determine the location of transmembrane alpha helix of a transmembrane protein. The algoritm takes a mRNA sequence and turns it into hydropathy indexes by amino acids. The subsequal distribution of hydropathy indexes is show by the sliding mean
The hydropathy indexes are based on the Kyte-Doolittle scale
